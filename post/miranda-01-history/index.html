Miranda的诞生

Miranda由Turner于1983年-1986年间开发，Miranda是一个non-strict，多态的，lazy的函数式编程语言，它直接影响了Haskell的设计。

与Miranda相关的文献

Richard S. Bird与Philip Wadler合著的《Introduction to functional programming》第一版的语言采用的即是Miranda。

Simon Peyton Jones的于1987年出版的《The Implementation of Functional Programming Languages》中也是借用了Miranda的语法，介绍函数式编程语言的实现，当然书中忽略了Miranda一些特性的，例如abstract data types 和 structured data types with laws。以及1991年的Implementing functional languages: a tutorial，这个tutorial对应书中的第二部分和第三部分。

Miranda的特性

使用布局，多态，模式匹配，柯里化，高阶函数，列表推导式，惰性求值，无限列表，强类型多态，自定义代数类型，抽象数据类型

Miranda与Haskell

在A History of Haskell: Being Lazy With Class[1]一文中第2.3和3.8节中有详细描述Haskell与Miranda的关系。
